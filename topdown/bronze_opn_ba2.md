**Program 1 - AN/SQQ-89 Surface ASW Combat System (Cost Codes DB400/DB830/DB900/DB6IN):**
- [line 2475] "The AN/SQQ-89(V) Surface Ship ASW Combat System provides integrated Undersea Warfare (USW) combat management, fire control, command and control, and on-board training to enable surface combatants to engage USW targets in both open ocean and littoral environments."
- [line 2477] "Variants of the AN/SQQ-89(V) are currently in operation on in-service CG47 and DDG51 Class Surface Combatants."
- [line 2484] Backfit platforms: "Select CG47 Class Baseline 3 and 4 (CGs 59-73) ships budgeted under OPN BLI 0960 (Cruiser Modernization); DDG51 Class Flight I/II (DDGs 51-78) ships budgeted under OPN BLI 0900 (DDG Modernization) and OPN BLI 2136 (represented here); and DDG51 Class Flight IIA and onward via OPN BLI 2136 (for DDGs 79-132 represented here) and SCN BLI 2122 (DDG51 Class Forward Fit for DDGs 133 and follow)."
- [line 2489] AN/SQQ-89A(V)15 comprised of: Multi-Function Towed Array (MFTA), COTS hardware, hull/towed Sonar Echo Tracker Classifier, Acoustic Intercept fused data, Surface ASW Synthetic Trainer (SAST).
- [line 2505–2509] Year-to-year funding profile: 1st year = procurement; 2nd year = advance planning/yard design; 3rd year = installation.

**Program 2 - Aircraft Carrier Tactical Support Center (CV-TSC) (Cost Code DB301):**
- [line 2528–2538] AN/SQQ-34 CV-TSC delivers ASW and Surface Warfare (SUW) combat capability to CVN Tactical Action Officer and Carrier Strike Group. Ship-installed on CVNs. Integrates sensor data from MH-60R, MH-60S, P-8, MQ-4C.

**Program 3 - Sonar Dome Rubber Window (SDRW) (Cost Code DB401):**
- [line 2542] "AN/SQS-53 Sonar Dome Rubber Windows (SDRWs) are installed on all CG47 and DDG51 class ships." Emergency replacement program for pressurized rubber acoustic windows.

**Program 4 - USW Decision Support System (USW-DSS) (Cost Code DB601):**
- [line 2553–2567] USW-DSS is Navy Program of Record ASW C2 net-centric system supporting Carrier Strike Groups (CSGs). Installed on CSG platforms (CVNs, CGs, DDGs) and shore nodes. Processing software hosted on ISNS and CANES afloat; COTS hardware ashore.
- [line 2566] "USW-DSS procurements are installed on CSG platforms and supporting shore nodes via permanent Ship Alterations (SHIPALTs)."

**Sub-cost codes under DB400 (DDG51 Class SQQ-89A(V)15):**
- [line 2585] Item 1.1: NRE - technology refresh/replacement of legacy SQQ-89(V) equipment.
- [line 2590] Item 1.2: Engineering Changes (ECPs) including cyber mitigation for TI/ACB configurations.
- [line 2597] Item 1.3: Integrated Logistics Support (ILS) planning.
- [line 2605] Item 1.4: AEGIS Integration Events (AIE) - recurring engineering support for SQQ-89A(V)15 / AEGIS compatibility.
- [line 2609] Item 1.5: Combat Systems Ship Qualification Trials (CSSQTs) - at-sea exercises to verify installation per NAVSEA Instruction 90963.1C.

- **Line 48 (Fixed Surveillance)**: Entire justification classified. Understood to be SOSUS/fixed hydrophone infrastructure ashore.
